Getting started:

> Setup the database stored in the 'database' folder.

> Open lib/connection/DatabaseConnection.php, row 123 and setup the account info for your database connection.

> Start local server and open the index.php-file inside the 'public' folder. (GUI version)

> To use the API, see documentation.pdf or run the index.html-file inside the 'client' folder.
